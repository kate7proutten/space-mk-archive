# space-mk-archive

本仓库用于归档与发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定域名或具体网站，仅作为静态 HTML 资料的存储与分发用途。

## 项目简介

space-mk-archive 是一个纯前端静态归档仓库，主要用于存放过去或当前的 HTML 页面快照、小型演示页面或临时性网页素材。仓库本身不包含后端逻辑，所有内容均以 HTML 文件形式存在，可直接通过浏览器访问或部署至任意静态托管服务。

该仓库适合长期保存一些不再更新但仍需保留的页面，或用于记录网页版本变迁。

## 目录说明

```
space-mk-archive/
├── pages/          # 存放所有独立 HTML 页面
├── assets/         # 页面引用的公共资源（CSS、JS、图片等，可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个归档页面，命名规则以页面主题或时间戳为准，无统一格式要求。
- `assets/`：若页面引用公共样式或脚本，可统一放置于此；部分页面可能自带内联资源，不依赖此目录。

## 页面归档说明

- 每个页面均为完整独立的 HTML 文件，可脱离网络环境运行（若未引用外部资源）。
- 归档页面内容保持原始状态，不进行功能性修改或内容更新。
- 页面之间的链接、样式、脚本互不干扰，各自封装。
- 归档时间戳或版本信息可在页面内标注（非必须）。

## 维护说明

- 本仓库不定期更新，新增页面或替换旧版页面。
- 不保证所有页面兼容所有现代浏览器，但尽量保持基础可用性。
- 不提供在线演示或预览链接，使用者可自行克隆仓库后本地浏览。
- 如有页面需要移除或替换，请提交 Issue 或 Pull Request 说明理由。
